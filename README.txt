# Teste Automatizado - eBay Pokémon

Este projeto contém um teste automatizado desenvolvido utilizando **Cypress** e **Node.js**.

---

## Pré-requisitos

Antes de executar os testes, é necessário ter instalado:

1. **Node.js**  
   - Baixe e instale a versão LTS recomendada: [https://nodejs.org/](https://nodejs.org/)

2. **Cypress 12.17.4**  
   - Será instalado via npm após instalar as dependências do projeto.

---

## Passo a passo de instalação

1. Abra o terminal na pasta do projeto.

2. caso esteja usando o powershell digite isso Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass e depois siga o passo a passo 

3. digite npm install cypress@12.17.4 --save-dev para instalar 

4. digite npx cypress open para abrir
